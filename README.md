# CS 301 Graph Coloring Project

## Overview
This project implements and compares different algorithms for the Graph Coloring problem:
- **Brute Force Algorithm**: Exhaustive search for optimal solution
- **Heuristic Algorithm**: Greedy approach for faster approximation

## Problem Description
Given an undirected graph G(V, E) and a positive integer k, determine if the vertices can be colored using at most k colors such that no adjacent vertices share the same color.
